CSS Grid Layout

The CSS grid layout module offers a grid based system of rows and columns. Allows developers easily create complex web layouts. Just like the Flexbox, it consists of a grid container and grid items.

The column-gap property specifies the gap between columns of a grid. The row-gap specifies the gap between the rows of a grid. The gap property is a shorthand for row and column gap.

Lines between columns are called column Lines and lines between rows are called row lines. grind-column-start and grid-column-end specify where to start and end a grid item. grid-column property is the shorthand for both properties. grid-row-start and grid-row-end specifies where to start and end a grid item. grid-row is a shorthand for both properties.

Grid Tracks

rows and columns of a grid are defined with the grid or grid-template properties. The fr unit stands for fraction and it can be used to define grids like other CSS lengths. The place-content property is the shorthand property for the align-items and justify-content. The grid area property is a shorthand for grid-row-start,grid-column-start,grid-row-end,grid-column-end.
