CSS Outline

An outline is a line drawn around the element's border. It has the following propertites:outline-style,outline-width,outline-color,outline,outline-offset. Outlines differs from borders. It is drawn outside the element's borders. However, it's style property is similar to that of border's. The invert value for the outline-color property ensures the outline is visible regardless of the background. It's shorthand is similar to that of border's.

The outline-offset property adds a space between the outline and the border.
