CSS 2D Transforms

The transform property applies a 2D or 3D transformation to an element. Functions of this property include translate(),rotate(),scaleX()scaleY(),scale(),skewX(),skewY(),matrix().

The translate() function moves an element from its current position according to x-axis and y-axis parameters.

The rotate() function rotates an element clockwise or counter-clockwise,depending on the given degree.

The scale() function increases or decreases the size of an element.







The scaleX() function increases or decreases the width of an element.

The scaleY() function increases or decreases the height of an element.

The skewX() function skews an element along the x-axis by a given angle.

The skewY() function skews an element along the y-axis by the given angle.








The skew() function skews an element along the x and y axis









The martix() function combines all the 2D transform functions into one. It takes 6 functions:scaleX(),skewY(),skewx(),scaleY(),translateX(),translateY().
