CSS Image Centering

Images can be centered horizontally in 2 ways:

  1. It can be centered horizontally using margin:auto. When using this property, the width must be smaller than that of the page and display:block.




  2. It can also be centered using display:flex and justify-content:center. The width must also be smaller than the page. We put the image in a div container with the css properties.

Images can be centered vertically with display:flex. We also use a div element for this task. we use the align-items:center to vertically center the image. The height of the image must be smaller than the div container.
