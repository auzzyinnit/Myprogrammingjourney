CSS Image Filter Effects

The filter property is used to add visual effects to an element. Funtions of this filter include: blur(),brightness(),contrast(),drop-shadow(),grayscale(),hue-rotate(),invert(),opacity(),saturate(),sepia().

Original Image

The blur() function adds blur to an element.

The brightness() function adjusts the brightness of an element.

The contrast() function adjusts the contrast of the element.

The drop-shadow() function applies drop-shadow to an element.

The grayscale() function converts an image to grayscale.

The hue-rotate() function applies color rotation to an element.

The invert() function inverts the color of an image.

The opacity() function defines the opacity of an element.

The saturate() function adjusts the saturation or color intensity of an element.

The sepia() function makes an image browner(sepia)
