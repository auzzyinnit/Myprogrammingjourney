CSS Float

The CSSfloat property specifies how an object should float. The CSS clear property specifies what elements can float beside the cleared element and on which side. Normally,div elements are displayed ontop of eachother. However, if we specify the float property, we can line them next to eachother:

clear and clearfix

After using the float property and we want the next element to be underneath, we use the clear property. The clear property has 5 values: none,left,right,inherit,both.

clearfix hack

If a floated element is taller than its container, we use the overflow:auto is used to "fix" the issue. However, there is a mordern hack to this issue:

    .clearfix::after{
      content:"";
      clear:both;
