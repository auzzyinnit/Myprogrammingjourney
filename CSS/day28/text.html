CSS Text

CSS has a lot of properties for text formatting.

Color

The color property is used to set the color of the text. The background-color property defines the color behind the text.

Text Alignment and Direction

The text-align property is used to set the horizontal alignment of a text. It can either be left,right,centered or justified. The justify value stretches the text to ensure that each text has equal width and the left and right margins are straight. The text-align-last property specifies how to align the last line of a text.

The direction and unicode-bidi properties can be used to change the direction of an element.

Text Decoration

The text-decoration-line property is used to add decoration line to text. You can combine more than one value. The text-decoration-color property is used to set the color of the decoration line. The text-decoration-style property is used to set the style for the decoration line. The text-decoration-thickness property is used to set the thickness of the text decoration. For the shorthand, only the text-decoration-line property is required. It is in the order of line-color-style-thickness. The text-decoration:none property is used to remove underlines from links as links are underlined by default.

Text Transformation

The text-transform property is used to specify uppercase and lowercase in a text. It can also capitalize the first letter of each word.

Text Spacing

The text-indent property is used to speify the indentation of the first line of a text. The letter-spacing property is used to specify the space between the characters of a text. The line-height property is used to specify the space between lines. The word-spacing property is used to specify the space between the words in a text. The white-space property specifies how white space is handled inside an element.

Text Shadow

The text-shadow property is used to add shadow to a text.
