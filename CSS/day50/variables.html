CSS Variables

The var() function is used to insert the value of a CSS variable. CSS variables can have a global or local scope. To create a variable with a global scope,declare it inside the :root selector, which matches the document's root element. To create a variable wuth a local scope, declare it inside it's selector. The var() function makes code easier to read and easier to change color values
