CSS Layout

The display property is the most important CSS property for controlling layout. It is used to specify how an element is shown on a web page. The default display value for most elements is block or inline.

Block elements always start on a new line and take up the full with of the line,e.g p,div while inline elements dont='t start on a new line and take up as much space as neccesary,e.g span,img. The display:none property is used to hide and show elements without deleting and recreating them. The script tag uses display:none as it's default property. The default display values can be overriden as well. This changes how the element is displayed only,not the type of element it is. So an inline element with it's display overriden cannot house another block element. The visibility:hidden property is used to hide elements.

Width and Max-width

Setting the width of a block element will prevent it from taking the full width available. The max-width property will improve the browser's handling of small windows. The browser then adds a horizontal scroll bar to the page. The marign:auto property horizontally centers the element in it's container

The position property

The position property specifies the type of positioning method to be used for an element. There are 5 different position values. They are fixed,relative,static,absolute,sticky..

An element with the position:static property is not positioned in any special way,usually according to the normal flow of the web page. They are not affected by the top,right,left and bottom properties.

An element with the position:relative property is positioned relative to its normal position. They can be positioned by the top ,left,bottom,right properties.

An element with the position:fixed property is position relative to viewport,meaning it stays the same place even though the page is scrolled. The top,right,bottom and left properties are used to position the element

An element with the position:absolute is positioned relative to the nearest positioned ancestor.

An element with the position:sticky is positioned based on the user's scroll position. It toggles between relative and fixed, depending on the scroll positiion. You must specify at least one of the top,right,left,bottom for sticky postioning.

You can position text over an image with the absolute property.
