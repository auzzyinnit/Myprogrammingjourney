CSS Backgrounds

CSS allows you to add multiple background images for an element via the background-image property. This entire section will have multiple background images. The background-size property allows you to specify the size of the background image. cover property of background-image scales the image to cover the entire content and as such,parts of the image may not be displayed. contain property ensures that the image stays within the boundaries of the content. If you want a background image that covers the entire background at all times, use the background:url('image') no-repeat center fixed; and background-size:cover; properties. The background-clip property is used to specify where the image is positioned. It has 3 values:border-box(outside edge of the border),padding-box(outside edge of the padding) and content-box(outside edge of the content). This goes for background-origin property as well.
