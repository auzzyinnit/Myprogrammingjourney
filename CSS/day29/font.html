CSS Fonts

Choosing the right font is important as it dictates the user's experience with the web page. In CSS, the font-family property is used to specify the font of a text.

Web Safe Fonts

These are fonts that are universally installed across all devices and browsers.This is why when choosing the font using the font-family property, add 2 or 3 more fonts and end with the generic font family. The best safe fonts include : Arial, Verdana, Tahoma, Times New Roman ,Brush Script MT,etc. The 5 generic font families are serif,sans-serif,monospace,cursive and fantasy.

Font Style

The font-style property is used style fonts. It is mostly used for italic text but it has 3 values;italic,normal and oblique. The font-weight property specifies the weight of a font. The font-variant property specifies whether or not a text should be displayed in a smal-caps font. The small-caps font raises the case of the lowercase letters but keeps them smaller than normal uppercase letters

Font Size

The font-size property specifies the size of the text. Setting font size with pixels gives you full control of the text size. Font size can also be increased with the em value. 1em equals 16px as that is the default size of a font. However, not all browsers support em properly. To fix this issue, combine percentage and em font size by setting the default size of text in the page to perentage. Now, rescaling doesn't affect the font size.

Google Fonts

If you do not want to use any of the standard fonts in HTML, you can use google fonts. They are free to use.

Google have over 1000 fonts to choose from. They can also be styled like regular HTML fonts. Google also have font effects that you can use.

Font Parings

Great font parings are essential for great design. Some rules for choosing font pairs include complement,choosing one boss, being from the same font super family and a bit of contrast as being too similar will conflict.

Font Properties

To use the shorthand property, it must be in the order of style,variant,weight,font-size/line-height and family. Font size and fanily values are required.
