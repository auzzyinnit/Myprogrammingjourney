CSS LInks

With CSS, links can be styled in different ways. They can be styled with any CSS property. Links can be styled according to what state they are in. The for states of links are : a:link,which is a normal unvisited link,a:visited,when the user has visited,a:hover,when the user mouses over it, and a:active, the moment the link is clicked.

Uncle Pele

a:hover must come after a:link and a:visited. a:active must come after a:hover.
