CSS Combinators

A combinator explains the relationship between selectors. There are 4 combinators in CSS:Descendant Combinator(space),Child Combinator(>),Next sibling Combinator(+) and Subsequent Sibling Combinator(~).

The desendant combinator matches all elements that are desendants of a specified element:

div p { color:blue }

Child combinator selects all elements that are children of a specified element:

div > p { color:blue }

The next sibling combinator is used to select an element that is directly after another specific element:

div + p { color:blue }

The subsequent sibling combinator is used to select all elements that are next siblings of a specified element:

div ~ p { color:blue }
