CSS Lists

There are two main types of lists... ordered lists(<ol>) and unordered lists (<ul>).

The list-style-type property allows you to specify the type of list item marker. Below is an unordered list with a square as the item marker and an ordered list with the lower roman as the item marker:

  • Rain
  • Sun
  • Autumn
  1. 6:16 in LA
  2. Euphoria
  3. Like That

Images as Item List Markers

The list-style-image property specifies an image as the list item marker.

  • Decateur
  • Seasons
  • Keep the family close

Position the list Item Markers

The list-style-positon specifies the position of the item-list markers(bullet points). The outside value means the bullet points will be outside the list. The inside value means the bullet points will be inside the list.

  • East Atlanta Love Letter
  • If You Are Reading This, It's Too Late
  • good kid,m.A.A.d city

The list-style-type:none property can be used to remove the markers/bullets. To remove the default margin and padding that comes with lists, set margin:0 and padding:0.

Shorthand Property

When using the shorthand property, the order of arrangement is type,position, and image.
