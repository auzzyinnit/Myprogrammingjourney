CSS Tables

The look of a HTML table can be greatly improved with CSS. To specify the border properties of a table in CSS, use the border property. If you need your table to span the entire width of the web page, add the width:100% to the <table> element. The border-collapse property is used to collapse the border to one single border, If you want borders only around the table, specify the border property only for the <table> element.

The width and height of a table are specified by the width and height properties. The text-align property sets the horizontal alignment of the content in <th> and <td>. To center align the content, add text-align:center. The vertical-align property sets the vertical alignment of the content in <th> and <td> elements.

Table Styling

To control the space between the border and content of a table, use the padding property on the <th> and <td> elements. For horizontal dividers, add the border-bottom property to the <th> and <td> elements. Use the :hover selector on <tr> to highlight table rows on mouse over. For striped tables, add the nth-child() selector and add background-color property to all even or odd table rows.

Responsive Table

A responsive table will add a horizontal scroll bar if the screen is too small to display the full content. To make a table responsive, add a container element (like div) with the overflow-x:auto property around the <table> element.
