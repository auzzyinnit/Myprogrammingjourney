CSS PSeudo-Elements

Pseudo-elements are used to style specific parts of an element e.g the first letter or list item markers.

The ::first-line pseudo-element

The ::first-line pseudo-element is used to add a special line to the first line of a text.It's properties include font properties,color properties,background properties,word spacing,letter spacing,text-decoration,vertical-align,text-transform,line-height,clear. This can only be applied to block elements.

The ::first-letter pseudo-element

The ::first-letter pseudo-element is used to style the first letter of an element. It also can only be applied to block level elements.

Pseudo-elements can be combined with HTML elements.

::before and ::after pseudo-elements

The ::before pseudo-element is used to insert some content before the content of the element, while the ::after pseudo-element is used to insert some content after the content of the element

The ::marker pseudo-element

The ::marker pseudo-element is used to select the markers of list items,e.g

  1. 10 feet
  2. 9 feet
  3. 8 feet

The ::selection pseudo-element

The ::selection pseudo-element matches the portion of an element that is selected by the user
