It is possible to style html elements that have specific attributes or attribute values. the [attribute] selector is select elements with specific attributes. Below, I will style all span elements with a title to be blue texted and red background color

The [attribute="value"] selector is used to select elements with a specified attribute and value. The [attribute~="value"] selector is used to select elements containing a specified word. The [attribute|="value"] selector is used to select elements with the specified attribute, whose value can be exactly the specified value, or the specified value followed by hyphen. The value has to be a word or words separated by hyphen. The [attribute^="value"] selector is used to select elements with the specified attribute, whose value starts with the specified value. The [attribute$="value"] selector is used to select elements whose attributes value end with a specified value. The value does not have to be a word. The [attribute*="value"] selector is used to select elements whose attributes contain a specified value.
