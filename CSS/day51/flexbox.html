CSS Flexbox

Flexbox is short for Flexible Box layout module. It is a layout method used for arranging items in rows and columns. It makes it easier to design a flexible responsive layout without using float or positioning. Flexbox is one-dimensional(rows or columns), while grid is 2-dimensional(grind and column). A flexbox always consists of a container <div> element and the items <div>

The container becomes flexible by setting display to flex. The flex-direction property specifies the display-direction of flex items in the flex container. row-reverse and column-reverse flips the direction of row and column. flex-wrap property specfies wether flex items should wrap or not. nowrap is the default. flex-flow property is a shorthand for flex-wrap and flex-direction. The justify-content property is used to align items horizontally when they do not use all the space available. The align-items property is used to align items vertically when they do not use all the space available. The values of justify-content and align-items are center,flex-start,flex-end,stretch,etc. The align-content property is similar to the align-item property but it adjusts the flex-lines instead of the flex-items. To perfectly center an item, set align-item and justify-content to center.

CSS Flex Items

The direct child elements of a flex container automatically become the flex items. The order property specifies the order of the flex items of the flex container. The default value is 0. The flex-grow property specifies how much a flex item will grow relative to the rest of the flex items.The default value is 0. The flex-shrink property specifies how much a flex item will shrink compared to the rest of the flex items. Default value is 1. The flex-basis property specifies the initial length of a flex item. The flex property is a short hand for flex-grow,flex-shrink,flex-basis.The align-self property specifies the alignment of a particular element in the flex container.
