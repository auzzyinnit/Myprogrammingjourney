CSS Counters

Counters let you adjust the appearance of content based on its placement in the document. CSS counters are like variables. The values can be incremented by css rules.

counter-reset creates a counter or resets one. counter-increment increments a counter value. content inserts generated content. counter() and counters() adds the value of a counter to an element. Counters can be nested as well.
