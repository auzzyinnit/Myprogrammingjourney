CSS Selectors

A selector selects the html element you want to style. There are 5 catergories of selectors :

  1. Simple Selectors-name,id and class
  2. Combinator Selectors-selects element based on a specific relationship between them.
  3. Pseudo-class Selectors-selects elements based on a certain state.
  4. Pseudo-elements Selectors-select and style part of an element
  5. Attribute Selectors-attributes or attribute values
