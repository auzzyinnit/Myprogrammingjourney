How to Add CSS

There are 3 ways to insert CSS in a web page, these are external css, internal css and inline css.

External CSS

With an external style sheet, you can change the entire look of a web page by changing just one file. This page, for example, was styled by an external CSS file.

Internal CSS

Internal style sheet can be used for one page and is defined inside the style element,located in the head tag.

Inline CSS

This can be used to style a single element

Multiple Style Sheets

Assume that a particular element was styled both on the external style sheet and internal style sheet; If the link tag was placed before the style tag in the head section, the inline CSS would be used. If the link tag was placed after, the external CSS would be used. In terms of the cascading order, inline css is always the highest.
