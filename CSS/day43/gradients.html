CSS Gradients

CSS gradients lets you display smooth transitions between 2 or more colors. There are linear gradients,radial gradients and conic gradients.

Linear Gradients

The linear-gradient() function creates a linear gradient .Transparency can also be used. The repeating-linear-gradient() function can be used to repeat linear gradients.

Radial Gradients

The radial-gradient() function creates a radial gradient. A radial gradient is defined by its center.


Conic Gradients

The conic-gradient function creates a conic gradient. Add border-radius:50% to make the conic gradient look like a pie chart. The [at position] specifies the center position of the gradient and [from angle] rotates the gradient at a specific angle. repeating-conic-gradient is used to repeat the conic gradient.
