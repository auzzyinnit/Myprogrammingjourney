The !important Rule

The !important rule is used to add more importance to a property/value than normal in CSS. It can overrride all other previous styling rules for that property on that element.

Performance and Optimization

Efficient CSS helps your web page load faster and run more smoothly. some tips for better optimized CSS include:

  • Use simple selectors
  • Avoid universal Selector for styling
  • Avoid inline styles
  • Combine and minify CSS
  • Placing CSS in the head tag so it loads before the page is rendered.
  • Use shorthand properties
  • Cache your CSS

Accessibility Styling

Good CSS should help the user interact properly with the website. Good guide lines include:

  • Use visible focus indicators e.g :focus pseudo-class
  • Provide high contrast between text and background for readability
  • Avoid Styling that hides focus
  • Use semantic tags like nav and aside instead of using div for everything
  • Use prefers-reduced-motion @media to check if user has asked to reduce animations and transitions. Some users prefer less animations on screen.
