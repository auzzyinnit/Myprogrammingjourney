CSS Animations

CSS Allows animation of elements without Javascript. An animation lets an element change from one style to another. When the @keyframe property is specified,the animation will change styles at certain times. The animation must be bound to an element. The animation-duration property specifies how long the animation would take to complete. The animation-delay property specifies the delay for an animation to start. Negative values of animation-delay property will make the animation start as though it has been playing for "N" seconds. animation-iteration-count property specifies how many times the animation should run. animation-direction property specifies wether the animation should run foward or backward or in alternate cycles. animation-timing-function property specifies the speed curve of the animation. animation-fill-mode property specifies a style for the target element when animation is not playing. The animation shorthand property consists of animation-name,animation-duration,animation-timing-function,animation-delay,animation-iteration-count,animation-direction
