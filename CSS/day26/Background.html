CSS Background

CSS background properties are used to add background effects for elements.

Background Color

To set a background color, use the background-color property. The opacity attribute is used to set the transparency of an element.

Opacity 1

Background Image

To set a background image, use the background-image property. By default, the image repeats both horizontally and vertically until it covers the entire page. to prevent the image from repeating, the background-repeat:no-repeat property stops it from repeating. repeat-x and repeat-y sets the repeat to only horizontal and vertical respectively. The background-position property allows you to specify the position of the background iamge. The background-attachment property specifies wether or not the image scrolls with the page or is fixed(fixed/scroll values).

Background Shorthand

Instead of specifiing the background color,image,repeat,attachment and postion, you can merge all into one declaration. e.g

body{background:#AAAAAA url(pexels-sohi-807598.jpg) no-repeat left top}

It must be in the above order (color,image,repeat,attachment,position)
