Margins

Margins are used to create space around elements,outside of any defined border.

Margins-Individuals Sides

CSS has properties for specifying the margin for eadch side of an element. margin-top,margin-bottom,margin-left,margin right are the properties for specifying the margin for each side of an element. The margin property can also use the shorthand property.

The auto value horizontally centers the element within the container. Values can be inherited from the parent element with the "inherit" value.

CSS Margin Collapse

Sometimes 2 margins can collapse into a single margin. This only happens to the top and bottom margins. The top and the bottom margins are collapsed into one margin equal to the one largest in value.
