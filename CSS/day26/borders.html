Borders

The CSS border property allows you to specify the border style width and color of an element.

The border-style property allows you to specify the type of border the element should have.

A border can also have mixed styles

The border-width specifies the width of the border. you can set different border widths for different sides.

The border-color specifies the color of the four borders. You can also set specific colors for different borders.

Border Shorthand

You can merge the border style,width and color. You can also specify the individual properties for one side.

The border-radius property is used to add rounded borders to an element
