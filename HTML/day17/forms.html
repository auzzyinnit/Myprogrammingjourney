Forms

A form is used to collect user input. The form element is used to create forms

Input Element

The input element is the most used form element.

Label Element

The label element defines a label for many form elements.

Input type=text

This defines a single-line input field for text input.

Input type=radio

This defines a radio button. It lets users selects one of two or more limited options.

Input type=checkbox

This defines a checkbox. Allows the user select 0 or more limited number of choices.

Submit Button

input type=submit defines a button for submitting the form data to the handler. The form handler is specified in the action attribute in the form tag





Each input field must have a name attribute. If the name attribute is omitted, the input field will not be sent at all
