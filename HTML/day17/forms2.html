Form Attributes

The action attribute

The action attribute defines the action to be performed when the form is submitted. Usually, the form is sent to a file on the server when the user clicks the submit button.

The target attribute

The target attribute specifies where to display the response that is received after submitting the form. The target attribute can be _blank,_parent,_self,_top, with _self being it's default value

The method attribute

The method attribute specifies the html method to be used when submitting the form data. The form data can be sent as URL variables as HTTP post transactions.(method="get" or "post" respectively). Never use get to send sensitive data.

The Autocomplete Attribute

This attribute specifies wether or not a form should have the autocomplete on or off.

The novalidate attribute

This is a boolean attribute and it specifies wether or not the form data should be validated when submitted.
