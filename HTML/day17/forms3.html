Form Elements

Form elements include:input,label,select,textarena,button,fieldset,legend,datalist,output,option,optgroup.

Input Element

This is the most used element for obvious reasons

Label Element

This specifies label for several form elements. The for attribute of the label element should match the id of the input element to bind them together.

Select Element

This specifies a drop down list.

Option Element

This specifies an option that can be selected. Add the selected attribute to define a pre-selected option. The size attribute specifies the number of visible values. The multiple attribute allows the user select more than one value.

Textarea Element

This defines a multiline input field(a text area). The row attribute specifies the number of lines in a text area, while the cols attribute specifies the visible width of the text area.

Button Element

This specifies a clickable button.

Fieldset and Legend Elements

The fieldset element is used to group related data in a form.The legend element defines a caption for the fieldset element.

Datalist Element

This defines a list of pre-defined options for an input element. The list attribute of the input element must confer with the id attribute of the datalist element

Output Element

This defines the result of a calculation, for example, one performed by a script.
