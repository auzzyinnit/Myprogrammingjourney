Input Types

The different input types in HTML includes: button,checkbox,email,tel,color,date,file,datetime-local,hidden,image,month,number,submit,text,password,radio,range,reset,search,url,week.

text
defines a single-line text input field.
password
defines a password field
submit
defines a button for submitting form data to form handler
reset
defines a reset button that will reset all form values to their default values
radio
defines a radio button
checkbox
defines a checkbox
button
defines a button
color
used for input fields that should contain a color
date
used for input fields that should contain date. use the min and max attributes to add restrictions
date-time local
specifies a date and time input field, with no time zone
email
used for input fields that should contain email address
image
defines an image as a submit button. path to image is specified by the src attribute
file
defines a file select field and a "browse" button for file uploads
hidden
defines a hidden input field not visible to the user
month
allows the user to select month and year
number
defines a numeric input field. You can also set restrictions with the min and max attributes.
range
defines a control for entering a number whose exact value is not important(like a slider control). Restrictions can also be set
search
used for search fields
tel
used for input fields that should contain a phone number.
time
allows the user select a time(no time zone)
url
used for input fields that should contain url address
week
allows the user select week and year
