HTML Attributes

value
value specifies the initial value for an input field
readonly
This specifies the input field is read only, meaning it cannot be modified
disabled
this specifies that an input field should be disabled, meaning the field is unusable and unclickable.
size
this specifies the visible width, in characters, for an input field. The default value for size is 20
maxlength
This specifies the maximum number of characters allowed in an input field
min and max
min and max specify the minimum and maximum values for an input field. they work with number,range,date,date-time local,month,week and time
multiple
multiple specifies that the user is allowed to enter more than one value in an input field. it works with email and file.
pattern
pattern specifies a regular expression that the input field's value is checked against, when the form is submitted. this works for text,date,search,url,tel,email, and password.
placeholder
this specifies, usually, a short hint that describes the expected value of an input field
required
this specifies that an input field must be filled out before submitting the form. this can be used in text,search,url,email,password,date pickers,number,checkbox, radio and file
step
this specifies the legal number intervals for an input field, e.g if step="2", legal numbers could be -4,-2,0,2,4,6.
autofocus
this specifies that a particular input field gets focus when the page loads
height and width attributes
the height and width attributes specify the height and with of an input image element
list
this refers to datalist element that contains predefined options for an input element
autocomplete
this specifies wether or not an input field should have autocomplete on or off. the autocomplete attribute allows the browser predict the value,based on earlier entries
