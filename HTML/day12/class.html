The Class Attribute

The class attribute is often used to point to a class name in the style sheet

This paragraph was styled by the class attribute

The class attribute can be used with any html element and is case sensitive.

Class Syntax

To create a class, a period (.) followed by the name of the class,e.g .fish. This paragraph was styled by the class named "fish"

An element can belong to more than one class...,e.g, this paragraph belongs to the fire and ice class.

Class is JS

Class can be used by Javascript to perfotm specific tasks,usually through the "getElementByClassName()" method
