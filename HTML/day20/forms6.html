Input Form Attributes

These are form* attributes for the HTML input element.

form attribute
They specify where the input element belongs to. The value of this attribute must be equal to the id attribute of the form element it belongs to.
formaction attribute
this specifies the url of that file that will process the input when the form is submitted. It overrides the action attribute of the form element.
formenctype attribute
this specifies how the form-data should be encoded when submitted;only for forms with method="post". This overrides the enctype attribute of the form element. It works with submit and image input types
formmethod attribute
this defines the HTTP method for sending form data to the action URL;this overrides the method attribute of the form element. It works with submit and image input types
formtargetattribute
this specifies a name or keyword that indicates where to display the response that is recieved after submitting the form.
formnovalidatea attribute
this specifies that an input element should not be validated when submitted;it overrides the novalidate attribute of the form element. It works in the submit input type
