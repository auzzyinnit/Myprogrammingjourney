Semantic Elements

Semantic Elements are elements with meaning. examples include:article,header,nav,etc...

Non-semantic elements include div and span.

Section Element
Defines a section in a document,e.g

Nickelodeon

Nickelodeon has been one of the best channels, delivering quality shows for kids for over a decade.

Article Element
The article element defines independent, self contained content.e.g

Forbes

Forbes is dedicated to the lives of certain figures in out mordern world...They could be artists, billionares, kings, actors,etc.

Header Element
This represents a container for introductory content or a set of navigational links. Although you can have several head elements, it cannot be placed within a footer element,address or another header element.
Footer Element
This defines a footer for a document or section. You can also have multiple footer elements.
Nav Element
This defines a set of navigation links.
Aside Element
The aside element defines some content aside from the one it is placed in. It should be indirectly related to the surrounding content e.g

The key to typing faster is in muscle memory...so aim to build muscle memory as fast as possible.
