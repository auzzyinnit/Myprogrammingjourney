The kbd element

The kbd element is used to define keyboard input. The content inside is always displayed in monospace font. e.g You must workhard and stay consistent to achieve your goals.

The samp element

The samp element is used to define sample output from a computer program. The content inside is displayed in the monospace font. e.g Furnitures like the Table and Chair are essential when building a home.

The code element does not preserve line breaks unless put in a "pre" tag. e.g

    
        x=5;
        y=6;
        z=x+y;

The var element

The var element is typically used to define variables in programming or mathematical expressions.The content is usually displayed in italics e.g let the universal equation be E=mc2
