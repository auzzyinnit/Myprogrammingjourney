The Head Element

The head element contains the title,style,script,meta,link and base elements. It is placed between the body and the html tags.

Title element
Defines the title of the document. It is very important for search engine optimizations.
Style Element
This element is used to style info on the html page.
Link Element
This is used to relate the current file to an external html document.
Meta Element
This specifies the character set,page description,keywords,author and viewport settings.
Script Element
This is used to define client-side Javascripts,e.g

watch this..

Base Element
This specifies the base URL and/or the target for all the relative URLs in a page. It must have a href or target attribute or both. There can only be one base element in a document

Viewport

The viewport is the user's visible area of the webpage. It varies with the device i.e it will be smaller on a phone,compared to a computer screen.

In the meta tag,width=device-width sets the width of the page to follow the width of the device. The initial-scale=1.0 sets the initial zoom of the page when it is first loaded by the browser.
