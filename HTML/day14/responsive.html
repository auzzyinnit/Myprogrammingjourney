Responsive

Responsive Web Design is simply about making web pages that look good on all devices.

The CSS width property set to 100% allows the image to be responsive and scale up and down:

The image:max-width property will make the image scale down if it has to, but it will not scale above its original size.

The picture element allows you to define different images for different browser sizes

Responsive Text Size

Text can be set with the "vw" unit, which means viewport width. That way, the text follows the window size

Media Queries

Media Queries can define completely different styles for different browser sizes.
